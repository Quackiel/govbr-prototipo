# Reformulação do Portal Gov.br - Protótipo de Alta Fidelidade

Protótipo acadêmico em HTML, CSS e JavaScript puro, preparado para publicação no GitHub Pages.

## Proposta

A interface reorganiza serviços públicos digitais com foco em clareza, seriedade institucional, acessibilidade e redução de etapas. O cidadão encontra documentos, saúde, educação, trânsito, assistência social, agendamentos, previdência, impostos e segurança da conta em uma navegação centralizada.

## Direção visual

A composição foi refinada para se aproximar de um portal público real: predominância de branco e cinza, azul institucional, bordas discretas, menus em cascata, tabelas legíveis e ações diretas. O excesso de cards coloridos, gradientes e elementos decorativos foi removido.

## Como rodar

Abra o arquivo index.html no navegador. Também é possível usar uma extensão como Live Server no VS Code.

## Publicação no GitHub Pages

1. Envie os arquivos para um repositório no GitHub.
2. Acesse Settings > Pages.
3. Em Branch, selecione main e /root.
4. Salve e aguarde o link de publicação.
